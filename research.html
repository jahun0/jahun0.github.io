Continual Learning for Anomaly-Based Network IDS

Paper Code

This project studies multi-class drift in anomaly-based intrusion detection for defense networks. I evaluated continual learning strategies and deployment considerations that balance operational constraints with detection performance. The findings support guidance for maintaining IDS performance in isolated environments.

What I did

  • Designed drift scenarios and evaluation protocols for continual learning baselines
  • Implemented experiments and analyzed tradeoffs between accuracy and operational cost
  • Integrated results into a Defense and Security journal submission (2024)

Hyperautomation for Non-Combat Administrative Tasks

Paper Talk

I developed a reference architecture for hyperautomation that combines process mining, RPA/OCR, workflow orchestration, and data governance for manpower-constrained units. The design targets secure, on-prem deployments and provides a framework for task selection and operational readiness. This work received the Best Paper Award (2024).

What I did

  • Authored the design framework and evaluation criteria for administrative automation
  • Prototyped secure on-prem pipelines and drafted deployment SOPs and checklists
  • Conducted officer interviews to translate policy constraints into system requirements

VR SBIRT Training with Conversational AI

Paper Code

This project created a virtual reality SBIRT training environment with a conversational AI virtual patient for scalable clinical practice. The system integrates speech input and output with dialogue state tracking and policy control, enabling realistic training conversations. Results were reported at AIED 2023.

What I did

  • Built the intent/entity pipeline and dialogue state tracking for SBIRT conversations
  • Integrated speech I/O with Unity and a web interface for training delivery
  • Co-authored the AIED 2023 paper and ran formative user studies

Deep RL Offloading for Cloud Robotics Navigation

Thesis Code

I designed deep reinforcement learning policies for computation offloading in cloud robotics navigation. Using a Clearpath Jackal in ROS/Gazebo, I benchmarked DQN and Double-DQN policies against heuristic baselines under variable network conditions. The study quantified performance and robustness under constrained links.

What I did

  • Implemented offloading policies and network emulation for latency and bandwidth limits
  • Ran real-world and simulated experiments to measure navigation performance
  • Authored the M.S. thesis on situation-aware cloud robotics navigation (2023)
